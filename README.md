# Distributed-Web-Crawler-XGBoost
基于 Scrapy + Redis 构建分布式房源爬虫，采集 10 万 + 房产数据用于房价分析。实现分布式任务调度与负载均衡，爬取效率提升 3 倍；通过 UA 池与代理 IP 实现反爬，使用 XGBoost 构建房价预测模型，并结合 SHAP 实现模型可解释性分析与可视化，输出房价关键影响因素与业务洞察。


1.spider-master使用分布式scrpy-redis框架爬取网站房地产信息

2.data_wash负责对本地爬取信息数据进行数据预处理，数据清洗，数据特征工程构建，机器学习建模，shapf分析主要影响特性。分别对应1，2，3，4，5号py文件，tools是封装好的数据清洗函数用来对特殊信息（如文字表示的数据转为数据等）进行数据清洗。

3.需要下载好redis,和python编辑器即可运行
